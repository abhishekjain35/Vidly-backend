## General info
Vidly is a video rental application. This repository is the backend part of it and is made with NodeJS, express and MongoDB.
	
## Technologies
Project is created with:
* NodeJs: 12.14.0
* ExpressJs: ^4.17.1
* Mongoose: ^5.9.0
* and MongoDB
	
## Setup
1. Clone it
2. To run this project:

`
npm install
`

`
node index.js
`
